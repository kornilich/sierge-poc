### How to run it on your own machine

1. Install UV

   ```bash
   curl -Ls https://astral.sh/uv/install.sh | sh
   ```

2. Install the requirements

   ```bash
   uv pip install -e .
   ```

3. Run the app

   ```bash
   $ streamlit run streamlit_app.py
   ```
